////////////////////////////////////////////////////////////////
为Excel导入文本文件的VBA代码
////////////////////////////////////////////////////////////////

文件说明：
Module1.bas 是VBA代码。
Worksheet1.xlsm 是包含宏的Excel文件，Worksheet1.xlsx 不包含宏。
list.txt 是用于导入的文本文件列表
AIB_20131030.txt, BOI_20131030.txt, Ulster_20131030.txt 是数据文件。

使用步骤：
1. 用Excel 2010打开Worksheet1.xlsm，启用宏。
2. 选中Sheet1中的文件列表（或者选中其中文件列表其中一个格也可以）。
3. 运行宏，就会为每个文件创建一个tab，读入文件内容到该tab，并将tab改名为文件名+当前时间。

注意事项：
由于在VBA代码中使用了Excel文件的路径，数据文件与Excel文件必须放在相同目录下。
文件名是从选定的表格区域中读取的，因此请在运行宏之前请先选择文件列表（选中整个文件列表或者部分Cell均可）。
