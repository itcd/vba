导入文本文件到Excel表格的VBA代码

文件说明：
Worksheet.xlsm 是包含宏（VBA代码）的Excel文件。
AIB_20131030.txt, BOI_20131030.txt, Ulster_20131030.txt 是数据文件。
LoopThroughFiles.bas 是单独的VBA代码文件（仅作备份之用）。

使用步骤：
1. 使用Excel 2010打开Worksheet.xlsm，启用宏。
2. 运行宏，Worksheet.xlsm所在目录下的所有文本文件（*.txt）就会被导入到以其文件名+当前时间命名的sheet中。
